Documentação da API do NeurotrackApp

Esta seção contém a documentação detalhada da API do NeurotrackApp, incluindo endpoints, parâmetros, respostas e exemplos de uso.

Visão Geral da API

A API do NeurotrackApp segue os princípios REST e utiliza JSON para comunicação. Todos os endpoints requerem autenticação via token JWT, exceto os endpoints de autenticação.

Base URL

https://api.neurotrackapp.com/v1

Autenticação

A autenticação é realizada através de tokens JWT. Para obter um token, utilize o endpoint /auth/login.

Exemplo de Requisição


POST /auth/login
Content-Type: application/json

{
  "email": "usuario@exemplo.com",
  "password": "senha123"
}

Exemplo de Resposta


{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "refreshToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "expiresIn": 3600
}

Endpoints Principais

Autenticação

  • POST /auth/register - Cadastro de novo usuário
  • POST /auth/login - Autenticação de usuário
  • POST /auth/google-login - Login com Google
  • POST /auth/logout - Logout de usuário

Usuários

  • GET /users/profile - Obter perfil do usuário
  • PUT /users/profile - Atualizar perfil do usuário
  • PUT /users/neuro-profile - Atualizar perfil neurodivergente
  • PUT /users/preferences - Atualizar preferências

Entradas Diárias

  • GET /dailyentries - Listar entradas diárias
  • POST /dailyentries - Criar nova entrada
  • GET /dailyentries/{id} - Obter entrada específica
  • PUT /dailyentries/{id} - Atualizar entrada
  • DELETE /dailyentries/{id} - Excluir entrada

Tarefas

  • GET /tasks - Listar tarefas
  • POST /tasks - Criar nova tarefa
  • GET /tasks/{id} - Obter tarefa específica
  • PATCH /tasks/{id} - Atualizar tarefa
  • DELETE /tasks/{id} - Excluir tarefa

Medicações

  • GET /medications - Listar medicações
  • POST /medications - Adicionar nova medicação
  • GET /medications/{id} - Obter medicação específica
  • PUT /medications/{id} - Atualizar medicação
  • DELETE /medications/{id} - Excluir medicação
  • POST /medications/{id}/log - Registrar uso de medicação

Agenda

  • GET /agenda - Listar eventos da agenda
  • POST /agenda - Adicionar novo evento
  • GET /agenda/{id} - Obter evento específico
  • PUT /agenda/{id} - Atualizar evento
  • DELETE /agenda/{id} - Excluir evento
  • POST /agenda/sync - Sincronizar com Google Calendar

Inteligência Artificial

  • POST /ai/generate-response - Interagir com assistente IA
  • POST /ai/summarize-tasks - Obter resumo de tarefas via IA

Saúde da API

  • GET /health - Verificar status da API
