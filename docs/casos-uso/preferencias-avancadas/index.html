Preferências Avançadas

O módulo de Preferências Avançadas do NeurotrackApp permite a personalização profunda da experiência do usuário, adaptando a interface e interações às necessidades específicas de pessoas neurodivergentes.

Principais Funcionalidades

Interface e Acessibilidade

  • Tema e cores: Personalização visual para reduzir sobrecarga sensorial
  • Tamanho de texto: Ajuste de fontes para melhor legibilidade
  • Densidade de informação: Controle sobre quantidade de elementos exibidos
  • Animações: Configuração de efeitos visuais e transições

Notificações e Comunicação

  • Estilo de notificações: Personalização de alertas (visual, sonoro, tátil)
  • Frequência: Controle sobre quantidade e agrupamento de notificações
  • Horários silenciosos: Períodos sem interrupções
  • Linguagem: Ajuste do tom e complexidade das mensagens

Personalização Cognitiva

  • Nível de detalhamento: Ajuste da quantidade de informações apresentadas
  • Estrutura de tarefas: Preferências para divisão e apresentação de atividades
  • Lembretes contextuais: Configuração de gatilhos e frequência
  • Perfil neurodivergente: Ajustes específicos para TDAH, TEA e outras condições

Cenários de Uso

Configuração de Interface para TDAH

  1. Usuário acessa preferências avançadas
  2. Seleciona perfil TDAH como base
  3. Ajusta densidade de informação para reduzida
  4. Configura cores de alto contraste para elementos importantes
  5. Ativa modo de foco que reduz distrações visuais

Personalização de Notificações

  1. Usuário acessa configurações de notificações
  2. Define horários silenciosos durante períodos de concentração
  3. Configura estilo de notificação preferido (vibração suave)
  4. Ajusta frequência e agrupamento de alertas
  5. Configura linguagem simplificada para mensagens
