Configurações de Segurança

O módulo de Configurações de Segurança do NeurotrackApp permite o gerenciamento de autenticação, privacidade e dados pessoais, garantindo proteção e controle para usuários neurodivergentes.

Diagrama de Fluxo

Principais Funcionalidades

Autenticação e Acesso

  • Gerenciamento de senhas: Alteração e recuperação de credenciais
  • Autenticação de dois fatores: Configuração de verificação adicional
  • Biometria: Configuração de acesso por impressão digital ou reconhecimento facial
  • Histórico de login: Visualização de acessos recentes

Privacidade e Compartilhamento

  • Controle de dados: Configuração de quais dados são coletados
  • Compartilhamento: Gerenciamento de permissões para profissionais e cuidadores
  • Exportação de dados: Obtenção de cópia dos dados pessoais

Segurança de Conta

  • Dispositivos conectados: Gerenciamento de dispositivos autorizados
  • Notificações de segurança: Alertas sobre atividades suspeitas
  • Exclusão de conta: Processo para remoção de dados

Endpoints da API

Método Endpoint Descrição
GET /security/settings Obter configurações de segurança
PUT /security/settings Atualizar configurações de segurança
POST /security/2fa/enable Ativar autenticação de dois fatores
POST /security/2fa/disable Desativar autenticação de dois fatores
GET /security/devices Listar dispositivos conectados
DELETE /security/devices/{id} Revogar acesso de dispositivo
GET /security/login-history Obter histórico de login
POST /security/data-export Solicitar exportação de dados

Cenários de Uso

Configuração de Autenticação de Dois Fatores

  1. Usuário acessa configurações de segurança
  2. Seleciona opção de ativar autenticação de dois fatores
  3. Escolhe método preferido (SMS, e-mail, aplicativo autenticador)
  4. Configura e verifica o segundo fator
  5. Recebe códigos de backup para emergências

Gerenciamento de Compartilhamento

  1. Usuário acessa configurações de privacidade
  2. Adiciona profissional de saúde como contato autorizado
  3. Configura quais dados podem ser compartilhados
  4. Define período de acesso
  5. Gera link de compartilhamento ou convite

Exportação de Dados Pessoais

  1. Usuário solicita exportação de dados
  2. Seleciona categorias de dados a incluir
  3. Escolhe formato de exportação
  4. Confirma identidade via senha ou segundo fator
  5. Recebe arquivo com dados pessoais
