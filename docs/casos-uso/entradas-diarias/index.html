Entradas Diárias

O módulo de Entradas Diárias do NeurotrackApp permite o registro de humor, atividades e observações diárias, ajudando a identificar padrões e gatilhos para pessoas neurodivergentes.

Diagrama de Fluxo

Principais Funcionalidades

Registro de Humor

  • Escala de humor: Registro visual e intuitivo do estado emocional
  • Gatilhos emocionais: Identificação de fatores que influenciam o humor
  • Histórico de humor: Visualização de tendências ao longo do tempo

Registro de Atividades

  • Atividades diárias: Registro de rotinas e eventos significativos
  • Nível de energia: Acompanhamento dos níveis de energia ao longo do dia
  • Produtividade: Avaliação subjetiva da produtividade em diferentes atividades

Observações e Notas

  • Notas livres: Espaço para registro de pensamentos e observações
  • Sintomas: Registro de sintomas físicos ou mentais
  • Medicações: Registro de medicações tomadas e seus efeitos

Endpoints da API

Método Endpoint Descrição
GET /dailyentries Listar entradas diárias
POST /dailyentries Criar nova entrada
GET /dailyentries/{id} Obter entrada específica
PUT /dailyentries/{id} Atualizar entrada
DELETE /dailyentries/{id} Excluir entrada
GET /dailyentries/stats Obter estatísticas de entradas

Cenários de Uso

Registro de Humor Diário

  1. Usuário acessa a tela de entradas diárias
  2. Seleciona o nível de humor na escala visual
  3. Registra fatores que influenciaram o humor
  4. Adiciona notas ou observações relevantes
  5. Salva a entrada e recebe confirmação

Análise de Padrões

  1. Usuário acessa a visualização de histórico
  2. Seleciona período de tempo para análise
  3. Visualiza gráficos de humor e energia
  4. Identifica padrões e correlações

Compartilhamento com Profissionais

  1. Usuário seleciona entradas relevantes
  2. Gera relatório para compartilhamento
  3. Envia relatório para profissional de saúde
