Resumo e Home

O módulo de Resumo e Home do NeurotrackApp oferece um dashboard personalizado com visão consolidada das informações, servindo como ponto central de acesso para pessoas neurodivergentes.

Diagrama de Fluxo

Principais Funcionalidades

Dashboard Personalizado

  • Visão consolidada: Resumo das informações mais relevantes
  • Widgets configuráveis: Elementos que podem ser personalizados
  • Priorização inteligente: Destaque para informações mais importantes
  • Adaptação contextual: Conteúdo que se ajusta ao momento e necessidades

Resumo de Atividades

  • Tarefas do dia: Visualização das atividades prioritárias
  • Compromissos próximos: Lembretes de eventos agendados
  • Medicações: Alertas de medicamentos a serem tomados
  • Progresso diário: Acompanhamento de metas e conclusões

Insights e Sugestões

  • Recomendações personalizadas: Sugestões baseadas em padrões
  • Lembretes contextuais: Avisos adaptados ao momento
  • Dicas de bem-estar: Sugestões para melhorar qualidade de vida

Endpoints da API

Método Endpoint Descrição
GET /dashboard Obter dados do dashboard
GET /dashboard/widgets Obter widgets disponíveis
PUT /dashboard/widgets Atualizar configuração de widgets
GET /dashboard/daily-summary Obter resumo diário
GET /dashboard/upcoming Obter atividades próximas
GET /dashboard/insights Obter insights personalizados
GET /dashboard/progress Obter progresso de metas

Cenários de Uso

Início do Dia

  1. Usuário abre o aplicativo pela manhã
  2. Visualiza resumo das tarefas e compromissos do dia
  3. Recebe lembretes de medicações matinais
  4. Verifica previsão de carga cognitiva para o dia
  5. Acessa sugestões de priorização

Personalização do Dashboard

  1. Usuário acessa configurações do dashboard
  2. Seleciona widgets relevantes para suas necessidades
  3. Organiza posição e tamanho dos elementos
  4. Define prioridades de informação
  5. Salva configuração personalizada

Verificação de Progresso

  1. Usuário acessa dashboard ao final do dia
  2. Visualiza progresso nas tarefas planejadas
  3. Verifica registro de medicações e atividades
  4. Recebe insights sobre padrões do dia
  5. Visualiza preparação para o dia seguinte
