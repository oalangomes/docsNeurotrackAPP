Análises e Relatórios

O módulo de Análises e Relatórios do NeurotrackApp oferece visualizações e insights sobre dados coletados, ajudando pessoas neurodivergentes a compreender seus padrões e evolução.

Diagrama de Fluxo

Principais Funcionalidades

Visualizações Interativas

  • Gráficos de humor: Visualização de tendências emocionais
  • Mapas de produtividade: Identificação de períodos de maior eficiência
  • Correlações: Visualização de relações entre diferentes variáveis
  • Linha do tempo: Evolução de métricas ao longo do tempo

Relatórios Personalizados

  • Relatórios periódicos: Resumos diários, semanais e mensais
  • Relatórios temáticos: Foco em aspectos específicos (medicação, tarefas, humor)
  • Exportação: Geração de PDFs e documentos compartilháveis

Insights e Recomendações

  • Padrões identificados: Detecção automática de padrões relevantes
  • Sugestões baseadas em dados: Recomendações personalizadas
  • Alertas preventivos: Identificação precoce de tendências negativas

Endpoints da API

Método Endpoint Descrição
GET /analytics/mood Obter análise de humor
GET /analytics/productivity Obter análise de produtividade
GET /analytics/medication Obter análise de medicação
GET /analytics/tasks Obter análise de tarefas
GET /analytics/correlations Obter correlações entre variáveis
GET /reports/generate Gerar relatório personalizado
GET /reports/export Exportar relatório em formato específico
GET /insights Obter insights personalizados

Cenários de Uso

Análise de Padrões de Humor

  1. Usuário acessa seção de análises
  2. Seleciona visualização de humor
  3. Define período de análise
  4. Visualiza gráficos e tendências
  5. Explora correlações com outros fatores (medicação, sono, atividades)

Geração de Relatório para Profissional

  1. Usuário seleciona opção de relatório para profissional
  2. Escolhe período e métricas a incluir
  3. Sistema gera relatório formatado
  4. Usuário revisa e exporta em PDF
  5. Compartilha com profissional de saúde

Exploração de Insights

  1. Sistema identifica padrão relevante nos dados
  2. Notifica usuário sobre o insight
  3. Usuário explora visualização detalhada
  4. Sistema sugere possíveis ações baseadas no insight
